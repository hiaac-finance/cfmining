class cfmining.algorithms.MAPOCAM(action_set, pivot, classifier, max_changes=3, compare=None, clean_suboptimal=False, warm_solutions=None, recursive=False)[source]¶

Class that finds a set of Pareto-optimal counterfactual antecedents using branch-and-bound based tree search.

Parameters

  • action_set (ActionSet type class,) – Contains the discretization of the features to find the counterfactual antecedents.

  • pivot (numpy type array,) – Sample that had an undesired prediction that we want to find counterfactual antecedents that change the outcome.

  • classifier (predictor.Classifier type class) – Wrapper to help on finding the counterfactual antecedent.

  • max_changes (int, optional (default=3)) – Maximal number of changes of a counterfactual antecedent.

  • compare (criteria type class.) – Class that evaluate the cost of a counterfactual antecedent.

solutions¶

Set of Pareto-optimal counterfactual antecedent.

Type

set of numpy arrays.
