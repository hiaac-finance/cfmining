class cfmining.action_set.ActionSet(X, names=None, **kwargs)[source]¶

Container of ActionElement for each variable in a dataset.

Parameters

  • df – pandas.DataFrame containing features as columns and samples as rows (must contain at least 1 row and 1 column)

  • X – numpy matrix containing features as columns and samples as rows (must contain at least 1 row and 1 column)

  • names – list of strings containing variable names when X is array-like

  • custom_bounds – dictionary of custom bounds

  • default_bounds – tuple containing information for default bounds - (lb, ub, type) where type = ‘percentile’ or ‘absolute’; - (lb, ub) if type is omitted, it is assumed to be ‘absolute’

  • default_step_type –

  • print_flag – set to True to print a table with the ActionSet as _repr_

  • check_flag – set to True to check for internal errors

class cfmining.action_set._ActionElement(name, values, bounds=None, variable_type=None, mutable=True, splits=None, step_type='relative', step_direction=0, step_size=0.01)[source]¶

Internal class to store the properties of actions in each dimension. Used by ActionSet.

Represent and manipulate feasible actions for a single feature

Parameters

  • name – name of the variable (at least 1 character)

  • values – values of the variable (must be non-empty, non-nan, finite)

  • bounds – bounds (must be a tuple of the form (lb, ub) or (lb, ub, bound_type) where bound_type is a valid bound type

  • variable_type – ‘int’ / ‘float’ / set to None (default) to determine automatically from values

  • step_direction – +1 or -1 or variable can only increase or decrease

  • immutable – boolean to represent whether or not the variable can be changed (True by default)
